# Dataset: Online Retail II

**Source:** UCI Machine Learning Repository  
**URL:** https://archive.ics.uci.edu/dataset/502/online+retail+ii  
**Direct download (Kaggle mirror):** https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci

## About the dataset
- ~1 million transactions from a UK-based online gift retailer
- Date range: December 2009 – December 2011
- Fields: Invoice, StockCode, Description, Quantity, InvoiceDate, Price, Customer ID, Country

## Setup instructions
1. Download `online_retail_II.xlsx` from either link above
2. Place the file in this `/data/` folder
3. **Do not commit the raw data file to GitHub** (it's in .gitignore)
4. The clean processed version `online_retail_clean.csv` is generated by `01_eda.ipynb`
